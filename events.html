Scheduled Dates, Time and Venue


  • Date: 16th October, 2019

  • Time: 5.30pm-8.30pm

  • Venue: AG02, International Institute of Information Technology, Bhubaneswar

Agenda


  • Introduction [40 mins]:
    1. What is open-source? Why open-source?
    2. What is Git and GitHub? Explanation about other cloud based Git services.
    3. What are open-source programs? Familiarize them with the list of summer/winter, paid/unpaid open-source programs.
    4. Talk on Hacktoberfest.
      • What is hacktoberfest?
      • How it will benefit?
      • Sign in to hacktoberfest.

    • Get started with OSS [1 hour]:
    1. Show a demo of actually solving any random issue from an OSS organisation.
    2. How to find a beginner-friendly issue?
    3. How to start contributing?
    4. Talk by GSoC holders.

    Hands Onn:
    1. Understanding actually how things are done.
    2. Create your first pull request.

    Conclusion [15 mins]:
    1. Information on Winter Of Code (to be conducted in December)
    2. Summarizing the evening.

Rules


  1. To qualify for the official limited edition Hacktoberfest shirt, you must register and make four pull requests (PRs) between October 1-31 (in any time zone).
  2. PRs can be made to any public repo on GitHub, not only the ones with issues labeled Hacktoberfest.
  3. If a maintainer reports your pull request as spam or behavior not in line with the projectâ€™s code of conduct, you will be ineligible to participate.
  4. This year, the first 50,000 participants who successfully complete the challenge will earn a T-shirt. (Last year 46,088 earned a shirt!)
